# Sleep & Wellness Analysis

## 📊 Dataset

Este proyecto utiliza un dataset de **Teen Social Media Usage and Mental Health** de Kaggle: https://www.kaggle.com/datasets/sureshbeekhani/teen-social-media-usage-and-mental-health

**Nota:** Este dataset es **sintético** y fue generado con fines de **práctica** educativa. Los datos no representan casos reales de salud mental.

## ❓ Preguntas de Análisis

### 1. Descripción
- ¿Cómo es el dataset?
- ¿Cuántas horas duermen en promedio? ¿Y el mínimo y máximo?
- ¿Qué plataforma de redes sociales es la más popular?
- ¿Cómo se distribuyen los niveles de estrés y ansiedad?
- ¿Cuántas personas tienen alto, medio y bajo riesgo de depresión?

### 2. Comparación
- ¿Los usuarios de TikTok duermen menos que los de YouTube o Instagram?
- ¿Hay diferencia en el nivel de estrés entre hombres y mujeres?
- ¿Quiénes tienen más interacción social tienen mejor rendimiento académico?
- ¿Las personas con alto riesgo de depresión hacen menos actividad física?

### 3. Correlación
- ¿Más pantalla antes de dormir significa menos horas de sueño?
- ¿A más horas en redes sociales, peor rendimiento académico?
- ¿Cuáles son las variables numéricas más correlacionadas entre sí?
- ¿Más actividad física implica menos ansiedad?

### 4. Predicción
- ¿Qué variables predicen mejor el riesgo de depresión?
- ¿Se puede predecir el nivel de estrés a partir del estilo de vida?
- ¿Qué perfil de usuario tiene más riesgo de depresión?
